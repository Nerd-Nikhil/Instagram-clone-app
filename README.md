# InstaClone - Full Stack Social Media Platform

InstaClone is a full-stack social media web application inspired by modern social networking platforms. The application enables users to create accounts, share posts, interact with content, and manage their profiles through a responsive and user-friendly interface.

The project was developed to understand scalable full-stack application architecture, REST API integration, authentication workflows, and efficient frontend-backend communication.

---

## Features

- User Authentication & Authorization
- Secure Login and Registration
- Create, Edit, and Delete Posts
- Like and Comment Functionality
- Responsive User Interface
- RESTful API Integration
- Real-Time User Interaction Experience
- Pagination, Filtering, and Sorting
- Profile Management
- Modular and Scalable Code Structure
